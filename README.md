# Food-Delivery-App-Dashboard-Project
End-to-end Power BI project focused on analyzing a food delivery app’s performance. Built using a star schema with a central fact table and related dimension tables, the dashboard provides insights into order trends, sales volume, and customer segments through interactive visualizations.
