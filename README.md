# Food-Delivery-App-Dashboard-Project
End-to-end Power BI project focused on analyzing a food delivery app’s performance. Built using a star schema with a central fact table and related dimension tables, the dashboard provides insights into order trends, customer behavior, and delivery outcomes through interactive visualizations.<br>
<br>
🗂️ Data Model<br>
**Fact Table**: Order ID, Date, Customer ID, Restaurant ID, Quantity, Delivery Status, Payment Method etc.<br>
**Customers Table**: Customer Name, Customer Status (Gold / Regular) etc. <br>
**Food Items Table**: Food Name, Restaurant Name,Restaurant Type etc.
